# 使用nvm进行Node版本控制

```bash
# 安装指定版本
nvm install v6.12.3

# 安装稳定版本
nvm install stable

# 删除指定版本
nvm uninstall ~

# 查看最近版本
nvm current

# 使用指定版本
nvm use ~

# 查看所有版本
nvm ls

# 设置默认版本
nvm alias default v8.10.0

# 升级npm为最新版
npm install npm@latest -g
```
